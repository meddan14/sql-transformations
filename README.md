# SQL Transformations & Challenges

A collection of SQL solutions focused on data engineering and marketing technology (MarTech) problems.

## Featured Project: Twitter Histogram of Tweets
**Problem:** Group users by their tweet frequency in 2022 and count how many users fall into each "bucket."

**Skills Used:** * Common Table Expressions (CTEs)
* Data Aggregation (COUNT, GROUP BY)
* Date Filtering (BETWEEN)
